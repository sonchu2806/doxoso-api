Đồng bộ từ máy bạn → Supabase

Luồng scrape chạy trên máy đang chạy lệnh node scripts/local-vietlott-bridge.js (IP nhà hoặc mạng của bạn), không qua Railway — thường qua được Cloudflare hơn.

Máy / điện thoại có phải bật không?
• Máy chạy Node (bridge): phải bật và giữ lệnh chạy trong lúc bạn bấm đồng bộ.
• Điện thoại (chỉ mở trang này qua WiFi để bấm nút): chỉ cần bật khi bạn đang thao tác; xong có thể tắt — dữ liệu đã lên Supabase.
• Nếu chỉ chạy bridge trên PC và bấm trên chính PC: chỉ cần PC bật lúc sync. 

Chưa chạy.

Cách dùng nhanh

  1. Trong thư mục doxoso-api, file .env có SUPABASE_URL và khóa (ưu tiên SUPABASE_SERVICE_ROLE_KEY).
  2. Chạy: npm run local-bridge
  3. Mở URL trên trình duyệt; điện thoại cùng WiFi dùng IP LAN máy tính thay cho 127.0.0.1.
  4. Lần đầu có thể chậm (Puppeteer / nhiều request).
