Đồng bộ dữ liệu hàng ngày

Chạy trên server Railway — máy tính/điện thoại của bạn không cần bật sau khi đã nhấn và thấy phản hồi JSON (chế độ nền). Dùng được trên điện thoại.

“Chờ xong” có thể 1–5+ phút; mạng di động/proxy đôi khi cắt kết nối trước khi xong.

Keno → Supabase (nhiều kỳ)

Sync hôm nay chỉ cập nhật khoảng “kỳ hiện tại” một lần. Để lấp các kỳ Keno thiếu trong DB, chạy Warm (lùi từ kỳ mới nhất, chỉ scrape kỳ chưa đủ số).

Backfill nặng hơn (ví dụ 1 ngày Keno): /admin/backfill-vietlott?products=keno&days=1 (nền; thêm &sync=1 khi chạy local để chờ JSON).

Đồng bộ từ máy nhà (local bridge)

Không thể thêm nút trên trang này để “chạy local”: trang admin chạy trên Railway, còn bridge phải chạy trên máy bạn (127.0.0.1). Nút ở đây chỉ gọi được API server, không mở được Node trên PC bạn.

Việc cần làm: clone/pull repo doxoso-api, đặt .env (Supabase), trong thư mục gốc chạy:

npm install
npm run local-bridge

Trên cùng máy đó mở trình duyệt: http://127.0.0.1:3847/ — điện thoại cùng WiFi dùng http://<IP-LAN-PC>:3847/. Tùy chọn LOCAL_BRIDGE_TOKEN trong .env rồi nhập token trên trang bridge.
